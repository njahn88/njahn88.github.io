    I have been able to connect two players in their own editor. I am using Network Objects and Network Transforms to relay information about the position and rotation of the players. While learning about Fishnet
    I ran into SyncVars, which opened the door to being able to update the players on both sides.
    Here is a snippet from my WeaponSystem.cs class, which currently handles the player switching weapons and firing weapons:


    The variable _currentIndex is a SyncVar of type integer. The line , _currentIndex.Value = currentWeaponIndex; , is updating the SycnVar variable and call the
    OnCurrentWeaponIndexChange function. The function goes through the players current weapons and finds which weapon should be set active based on the
