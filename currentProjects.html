    Update 1:
    I recently started the project and have been working on getting the Multiplayer systems setup. I have been doing a lot of research into the Fishnet Unity package.
    Since this is going to be a first person shooter of some sort there are several things that I still need to look into to make the game as smooth as possible.

    Goals:

    1: Learn Fishnet and included scripts for multiplayer setup
    2: Understand how lag compensation works and how to implement it with Fishnet
    3: Learn how to make overarching systems for the server to use to handle game states
    4: Deepdive into new game mechanics to make game different from what is already out there


    I have been able to connect two players in their own editor. I am using Network Objects and Network Transforms to relay information about the position and rotation of the players. While learning about Fishnet
    I ran into SyncVars, which opened the door to being able to update the players on both sides.
